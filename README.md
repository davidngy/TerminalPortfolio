Terminal Portfolio

Ein interaktives Portfolio im Terminal-Stil.

Die Idee hinter diesem Projekt war, ein persönlicheres Portfolio zu bauen. Klassische Webseiten fühlen sich oft gleich an. Da mein Interesse stärker im Backend und in Systemen liegt als im reinen Frontend-Design, wollte ich eine Umgebung schaffen, die sich mehr wie ein Werkzeug als wie eine typische Webseite anfühlt.

Das Ergebnis ist ein Terminal-inspiriertes Interface, in dem Navigation über Befehle funktioniert. Statt durch Menüs zu klicken, erkundet man das Portfolio über Commands wie in einer kleinen CLI.

Features

Terminal-ähnliche Benutzeroberfläche im Browser

Command-System mit Registry und modularen Commands

Projektanzeige mit strukturierten Karten

Minimalistisches Design mit Fokus auf Lesbarkeit

Persönliche Texte und Storytelling

Tech Stack

TypeScript
